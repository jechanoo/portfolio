## 第一个月策略

### 定价策略（重要！）
- ⭐ **前5单故意低价** — 平台上的竞争是靠评分，不是靠简介
- 目标：接到5单 → 拿5个5星好评 → 然后提价
- Basic 定价 $5-10 抢第一单
- 拿到3个好评后提到正常价

### 如何抢第一单
1. 每天花30分钟刷 Upwork 新发布的爬虫/数据录入需求
2. 前5单不要挑，小单也接（$5 的也做）
3. 响应速度比价格重要——需求发布15分钟内投递
4. Cover letter 短：只说 "I can do this. Here's a sample of similar work I've done: [作品网站链接]"
5. 做完后主动问客户 "Anything else you need?"（回头客比新客容易）

### 哪些单子最好接（新手友好）
- 数据录入 / 数据整理（Data Entry）— 可以用 Python 自动做
- 复制粘贴类任务 — 脚本搞定
- 简单网页抓取 — "scrape product list from X website"
- 文件格式转换 — CSV → Excel, PDF → Text 等
